# Adaptive Access Control using AI in Cloud Identity Management

An industry-level, AI-powered adaptive authentication and access control system simulating a real-world enterprise IAM platform. It dynamically scores login request risk (via a machine learning engine) and applies flexible, administrator-defined security policies in real time.

## Project Structure

```text
adaptive-access-control/
│
├── frontend/             # React.js + Tailwind CSS UI
├── backend/              # Node.js + Express.js APIs
├── ai-engine/            # Python + FastAPI Anomaly Detection Engine
└── README.md
```

## Technology Stack

- **Frontend**: React, Tailwind CSS, Vite, Axios, React Router, Chart.js
- **Backend**: Node.js, Express.js, MongoDB (Mongoose), JWT, bcryptjs
- **AI Service**: Python, FastAPI, Scikit-learn (Random Forest), Pandas, NumPy, Joblib
- **Authentication**: Stateful Context Analysis + Dynamic Policy Resolution + OTP Challenge
